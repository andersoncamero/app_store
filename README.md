# App Store

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

App Store es una tienda virtual diseñada para facilitar la venta y compra de productos. Ofrece una plataforma moderna, intuitiva y funcional donde los usuarios pueden publicar productos, explorar catálogos y realizar compras de manera eficiente.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Características principales:
- 🛍️ Publicación de productos: Los usuarios pueden listar productos con imágenes,          descripciones y precios personalizados.
- 🔍 Búsqueda y filtros avanzados: Encuentra productos fácilmente mediante categorías, palabras clave o filtros dinámicos.
- 💳 Carrito de compras y pagos: Gestiona tus compras con un carrito integrado y un proceso de pago fluido.
- 🧾 Gestión de pedidos: Realiza un seguimiento de las compras y ventas en tiempo real.
- ⭐ Reseñas y calificaciones: Los compradores pueden valorar productos y compartir sus opiniones.
- 📊 Panel de control: Ofrece métricas de ventas y estadísticas detalladas para los vendedores.
